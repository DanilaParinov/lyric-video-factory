# Lyric Video Factory

Сервис для автоматической генерации lyric video. Подаёте аудио, тексты с таймингами и набор видеоклипов — получаете N готовых вариантов в формате `.mp4`.

## Зависимости

- [FFmpeg](https://ffmpeg.org/download.html) — должен быть в `PATH` (`ffmpeg` и `ffprobe`)

## Быстрый старт

```
lyric-video-factory.exe
```

Откроется браузер с веб-интерфейсом. Там можно:
- загрузить или отредактировать шаблон таймингов
- собрать пул видеоклипов (перетащить файлы или папку)
- загрузить аудио и шрифт
- запустить генерацию и скачать результаты

## Структура рабочей директории

```
lyric-video-factory.exe
SF-Pro-Display-Thin.otf   ← шрифт по умолчанию (можно заменить)
template.json              ← пример шаблона
input/                     ← аудио и видеоклипы для автозагрузки в пул
output/                    ← результаты генерации (создаётся автоматически)
uploads/                   ← файлы загруженные через интерфейс (создаётся автоматически)
```

## CLI-режим

```
lyric-video-factory.exe template.json [N]
```

| Аргумент | Описание |
|---|---|
| `template.json` | Путь к шаблону (нативный или Creatomate) |
| `N` | Количество вариантов (по умолчанию 1) |

Результаты сохраняются в `output/variant_01.mp4`, `output/variant_02.mp4` и т.д.

## Формат шаблона

```json
{
  "width": 1080,
  "height": 1920,
  "audio": "input/song.wav",
  "dim_level": 0.3,
  "font": {
    "file": "SF-Pro-Display-Thin.otf",
    "size": 72,
    "color": "#ffffff"
  },
  "text_baseline_y": "50%",
  "texts": [
    { "text": "Первая строка",  "start": 1.2, "end": 3.8 },
    { "text": "Вторая строка", "start": 4.0, "end": 6.5 }
  ],
  "video_segments": [
    { "start": 0.0, "end": 3.8 },
    { "start": 3.8, "end": 6.5 }
  ]
}
```

| Поле | Описание |
|---|---|
| `width` / `height` | Разрешение выходного видео в пикселях |
| `audio` | Путь к аудиофайлу (опционально — обнаруживается из `input/` автоматически) |
| `dim_level` | Затемнение фона от `0.0` до `1.0` (0.3 = 30%) |
| `font.file` | Путь к файлу шрифта (`.otf` или `.ttf`) |
| `font.size` | Размер шрифта в пикселях |
| `font.color` | Цвет текста в формате `#rrggbb` |
| `text_baseline_y` | Вертикальная позиция базовой линии текста в процентах высоты кадра |
| `texts` | Массив строк лирики с таймингами в секундах |
| `video_segments` | Временны́е окна, под которые подбираются случайные клипы из пула |

### Поддержка шаблонов Creatomate

Шаблоны, экспортированные из [Creatomate](https://creatomate.com/), принимаются напрямую. Нужные поля извлекаются автоматически, недостающие заполняются значениями по умолчанию.

## Поддерживаемые форматы файлов

| Тип | Форматы |
|---|---|
| Видео (пул клипов) | `.mp4` `.mov` `.avi` `.mkv` `.webm` `.m4v` `.flv` `.wmv` |
| Аудио | `.wav` `.mp3` `.aac` `.m4a` `.ogg` `.flac` |
| Шрифт | `.otf` `.ttf` |
